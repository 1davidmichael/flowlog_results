Flowlog Upload
==============

This is a simple script to upload an example flowlog to a precreated S3 bucket.

Requirements
------------

* python 3
* virtualenv
* Pre-created S3 bucket
* AWS access and secret key with permission to write S3 object

Running
-------

Create virtualenv
```
make venv
```

Install dependencies
```
make reqs
```

Test
```
make test
```
